## The Product: Neo 1

![image.png](attachment:b2d06be1-e775-4ae7-95d3-ea833d1ea0a7:image.png)

Neo 1 is a pendant that:

- Passively records offline convos
- Transcribes with high-accuracy NLP
- Syncs to a mobile app where users can query what was said and when

**Core Features**

1. Always-on voice capture
2. Semantic search
3. Secure, encrypted storage
4. AI chat interface
5. Real-time calendar/task integrations (in roadmap)
